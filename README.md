## 📝 Usage 
1. Launch the game
2. Use your preferred injector and inject the DLL
3. Press CTRL+T to toggle the in-game GUI

## ✨ Features
#### Cheats
- No Skill Cost
- God Mode
- Dumb Enemies (enemies ignore you)
- One Hit Kill
- Instant Win
- Skip Battle Summary
- Player Stats Editor
- Timescale (in-battle speed)
- Skip Dialog

#### Quality of life
- Config
- Profiles
- Hotkeys
